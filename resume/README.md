# Resume / CV

El botón "Download Resume (PDF)" del hero (`index.html`) enlaza a:

```
resume/CV_CORDOVA.pdf
```

Para actualizar el CV en el futuro, sustituye ese archivo manteniendo
exactamente el mismo nombre — así no hace falta tocar el HTML — y haz:

```
git add -A && git commit -m "Update CV" && git push
```

Recomendación: que siga siendo de **una página** (estándar en banca/PE/
infraestructura).
